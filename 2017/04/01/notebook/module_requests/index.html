Requests模块摘要

2017-04-01
Python

summary: cookie部分内容未涉及，暂蒙圈，后补
requests:[.get, .put, .delete, .head, .options]
requests.get(url,params,headers,auth,cookies,timeout,proxies)
requests.post(url,data,files)
r:[.status_code, .text, .content, .cookies]

发送请求

import requests
r = requests.get(url)

除了[.get]也可以使用其他HTTP请求类型：[.put, .delete, .head, .options]

组合URL的传送

get实例

payload = {'key1': 'value1', 'key2': ['value2', 'value3']}
r=requests.get("http://httpbin.org/get",params=payload,headers=headers)
print(r.url)
# http://httpbin.org/get?key2=value2&key1=value1&key2=value3
r = request.get(url, auth=('usr', 'passwd'))   //身份认证

post实例

requests.post(url, data={'conment': 'test post'})
files = {'file': open('/home/123.md','rb')}
requests.post(url, files=files)

读取内容

r=requests.get(url)
r.text      //解码后返回的数据
r.encoding  //找到HTTP头部对响应的编码，也可自定义
r.content
r.status_code //检测响应状态码，比如200，404
r.raise_for_status() //对于4XX和5XX的响应抛出异常

读取二进制响应，比如图片

from PIL import Image
from io import BytesIO
i = Image.open(BytesIO(r.content))

对于JSON

r = requests.post(url, data=json.dumps({'some': 'data'}))
r.json()
