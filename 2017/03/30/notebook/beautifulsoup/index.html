BeautifulSoup文档摘要

2017-03-30
Python

Summary:
find:[find() find_all() find_parents() find_parent() find_next_siblings() find_previous_siblings() find_all_previous() find_previous()]
tag:[.name .attrs .contents .children .descendants .string(s)
.stripped_strings .parent(s) .next_sibling(s) .previous_sibling(s)
css选择器:[]
tag:[append() insert() insert_before() insert_after() clear() extract()]

1.创建对象

from bs4 import BeautifulSoup
bsObj = BeautifulSoup(html_doc, 'html.parser')

此处html_doc为html属性的文档

from urllib.request import urlopen
from bs4 import BeautifulSoup
html = urlopen('http://www.baidu.com')
bsObj = BeautifulSoup(html.read(), 'html.parser')

再比如：

bsObj = BeautifulSoup(open('index.html'))

2.调用对象，其中的内容都是可以想办法替换的，具体参见文档

soup.title
# <title>The Dormouse story</title>
soup.title.name    //可以对tag.name赋值改变文档内容
# u'title'
soup.title.string
# u'The Dormouses story'
soup.title.parent.name
# u'head'
soup.p
# <p class="title"><b>The Dormouses story</b></p>
soup.p['class']   //soup.tag.attrs操作类字典
# u'title'

3.查找对象

find()

soup.find('a', id='link3')

find_all()

for link in soup.find_all('a', class='sister'):
    print(link.get('href'))
for tag in soup.find_all(re.compile("^b"))
    print(tag.name)    
soup.find_all(href=re.compile('elsie'))
soup.find_all(['a', 'b'])
soup.find_all("a", limit=2)
soup.find_all(True)           //匹配所有
soup.find_all(function_name)  //也可匹配函数

strings

for string in soup.strings:
    print(repr(string))

4.从文档中获取所有文字内容

soup.get_text()

5.CSS选择器

soup.select('title')

通过标签逐层查找

soup.select('body a')

找到某个tag下的直接子标签

soup.select('head > title')

鉴于CSS语法尚未学习，该部分内容以后补上

5.修改文档

追加tag内容:

tag.append()          //把新元素添加到副节点.contents属性最后
tag.insert(1, "but did not endorse") //把元素插入到指定的位置
insert_before() insert_after() //在当前tag或文本节点前后插入内容

创建tag

new_tag = soup.new_tag('a', href='http://...')
original_tag.append(new_tag)

移除当前tag内容

tag.clear()注意不是删除当前的tag,而是把123 变成

tag.clear()

将当前tag移除文档树，并作为方法结果返回

extract()实际上产生两个文档树

i_tag = soup.i.extract()

将当前节点移除文档树并完全销毁

soup.i.decompose()

移除文档树中的某段内容,并用新tag或文本节点替代

new_tag = soup.new_tag("b")
new_tag.string = "example.net"
a_tag.i.replace_with(new_tag)

包装，给当前tag套一层父元素

wrap()

解包，移除tag内的所有tag标签

unwrap()
