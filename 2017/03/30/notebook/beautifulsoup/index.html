#BeautifulSoup学习概要记录
Summary:find:[find() find_all() find_parents() find_parent() find_next_siblings() find_previous_siblings() find_all_previous() find_previous()]
tag:[.name .attrs .contents .children .descendants .string(s)
.stripped_strings .parent(s) .next_sibling(s) .previous_sibling(s)
css选择器:[]
tag:[append() insert() insert_before() insert_after() clear() extract()]

##1.创建对象
from bs4 import BeautifulSoup
bsObj = BeautifulSoup(html_doc, ‘html.parser’)
此处html_doc为html属性的文档
from urllib.request import urlopen
from bs4 import BeautifulSoup
html = urlopen(‘http://www.baidu.com‘)
bsObj = BeautifulSoup(html.read(), ‘html.parser’)
再比如：
bsObj = BeautifulSoup(open(‘index.html’))

##2.调用对象，其中的内容都是可以想办法替换的，具体参见文档
soup.title

# <title>The Dormouse story</title>

soup.title.name 可以对tag.name赋值改变文档内容

# u'title'

soup.title.string

# u'The Dormouses story'

soup.title.parent.name

# u'head'

soup.p

# <p class="title"><b>The Dormouses story</b></p>

soup.p[‘class’] soup.tag.attrs操作类字典

# u'title'

tag.insert(1, “but did not endorse”)

###创建tag
new_tag = soup.new_tag(‘a’, href=’http://…’)
original_tag.append(new_tag)
